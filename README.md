# 大众点评 WebApp(React技术栈实现)

将大众点评主要的一些页面都实现了,使用webpack-dev-server模拟了登录功能

用到的技术栈有:

- react-router:路由跳转
- redux-saga:异步处理
- styled-jsx:css-in-js
- mockjs:mock数据
- axios:发起request
